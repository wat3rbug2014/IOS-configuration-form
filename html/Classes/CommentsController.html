CommentsController Class Reference

Inherits from UIViewController
Conforms to MFMailComposeViewControllerDelegate
Declared in CommentsController.h

Overview

This class of view controller allows the entry of comments so that extra notes can be used to describe the equipment changes. It also uses the MFMailComposeViewControllerDelegate so that the email can be created and sent out from this view.
