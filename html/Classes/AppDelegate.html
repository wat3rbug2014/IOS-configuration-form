removeRegionsFromMonitoring:

This method provides a means of cleaning up the locationmanager for the regions it monitors. The locationmanager is constrained to doing approximately 20 locations, so it is important to clean up prior instances. At this moment no overlap between other apps has been observed so this method makes the assumption that the regions were created by this app. The return type for [locationmanager monitoredRegions] is NSSet. For this reason the method accepts NSSet instead of NSArray.

- (void)removeRegionsFromMonitoring:(NSSet *)regions

Parameters

regions

A set of regions that will be removed.
