Inherits from BasicDeviceData : NSObject
Conforms to ConfigurationDataProtocol
Declared in AddDeviceData.h

Overview

This class extends the BasicDeviceData class so that it can be used in the AddDeviceViewController for adding devices. It is a data model to be used by the view controller.
